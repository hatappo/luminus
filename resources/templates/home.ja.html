Luminusは、軽量ライブラリのセットをベースにしたClojureのマイクロ・フレームワークです。 堅牢、スケーラブル、そしてプラットフォームとして使うのが簡単であることを目指しています。 Luminusなら、あなたが望むやり方で、気を散らされることなく、あなたのアプリケーションの開発に集中することができます。

ClojureがWeb開発をシンプルに

Quick Start Guide

$ lein new luminus my-app
$ cd my-app
$ lein run
Started server on port 3000

Leiningenを使って数秒で始めましょう

Developer Friendly

REPLと組み込みの開発用サーバで即座にハックを開始

  • 簡潔で容易に学べる
  • REPL駆動開発
  • パワフルなビルド・ツール

Productive

JVMを活用することによって生産性とパフォーマンスを獲得

  • JVMのトップに構築

  • 本番環境用にコンパイル

Modern Web & Mobile

Luminusは、WebやMobileのモダンなアプリケーション向けに構築された

  • ClojureScriptのサポート

  • 素晴らしい SQL/NoSQL のサポート
